Block PC1 from reaching the Server
On R2:

enable
configure terminal

access-list 10 deny 192.168.10.10 0.0.0.0
access-list 10 permit any

interface g0/1
 ip access-group 10 in

end
copy running-config startup-config


Verify Standard ACL:

show access-lists
show ip interface g0/1

Test Standard ACL
From PC1:
ping 172.16.1.100

Expected:
FAIL

Remove Standard ACL:
enable
configure terminal

interface g0/1
 no ip access-group 10 in

end
